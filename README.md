### Quickstart

* 가장 빠르게 Elastic Stack 환경을 구성하는 방법은 docker-compose 를 활용하는 것이다.
* 튜토리얼을 진행하는 개발 환경 내 docker 와 docker-compose 가 설치된 상태로 진행합니다.

```sh
$ docker-compose up
```
